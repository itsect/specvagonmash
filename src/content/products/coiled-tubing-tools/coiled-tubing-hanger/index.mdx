**Функции**

*  Разработаны для поддержки бурильных колонн колтюбинга, разработка в условиях понимания.
*  Прочный корпус подвески для определения работы при высоком увеличении в скважине.
*  Подходит для труб с наружным диаметром до 3 1/2 дюйма.
*  Фланцевый боковой выход предоставляется по требованию пользователя.
*  Доступны для всех фланцевых соединений API 6B и 6BX и рабочего давления.
*  Помещается переходником для ввода кабеля для направления канатных инструментов в отверстие НКТ. Этот переходный процесс повторяет кольцевое пространство и локализацию расположения НКТ вверх.
