**Подвески для трубок** – Подвески для трубок PARVEEN бывают четырех типов. Это тип оправки «PTHSM», тип с оберткой «PTHSW», тип с прикрепленной шейкой «PTHSE» и тип с расширенным разделением «PTHDE». Тип с удлинённой горловиной с линиями управления «PTHSCL», тип с расположенным разъемом с линиями управления «PTHDCL», тип оправки с вдавливаниями металл-металл (PTHTSM) и поправки тип о наличии ограниченной линии управления с вдавливаниями металл-металл, например, PTHTSMM. .

Подвеска НКТ типа 'PTHSM' используется для катушек НКТ типа PTHS, описанных ниже. Он герметизирует давление в круглом корпусе, сжимая его под нагрузкой НКТ и прижимных винтовок.

Тип расширенной шейки «PTHSE» используется для головок/золотников НКТ, описанных ниже. Этот тип похож на трубную подвеску типа PTHSM, но имеет усиленную горловину с уплотнительным кольцом, которое герметизирует жидкость в трубке. Этот тип герметизирует давление в круглом корпусе, сжимая его под нагрузкой НКТ и прижимных винтов.

Тип 'PTHSW' представляет собой трубную подвеску разъемного типа, которую можно обернуть вокруг трубки. Он герметизирует кольцевое давление, сжимая прижимными винтами, установленными на верхних фланцах золотника. Этот тип подвески используется там, где трубку необходимо вращать или перемещать вверх и вниз под давлением. Эта подвеска для труб используется вместе с муфтой для подвески труб и катушкой для подвески для труб.
