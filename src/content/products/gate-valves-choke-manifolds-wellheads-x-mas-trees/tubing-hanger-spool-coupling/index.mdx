PARVEEN также производит **катушки для трубных подвесок и** муфты для трубных подвесок. Катушка подвески для трубок используется в акустике с муфтой для подвески трубок, подвеской для труб с обертыванием (PTHSW) и катушкой для трубок типа 'PTHS'. Соединительная муфта подвески для трубы может быть использована путем завинчивания накидной гайки муфты подвески для трубы без необходимости поворота фонтанная арматура. Муфта имеет резьбу для установки обратного клапана. Верхние и низкие муфты могут иметь резьбу в соответствии с требованиями заказчика.
