**Крышка PARVEEN X-Mas Tree** - это верхний соединитель, который используется на верхней части X-Mas Tree. Его сфера деятельности - доступ к отверстию X-Mas Tree. Он в основном состоит из фланцевого корпуса, заглушки, ударной гайки в качестве основных частей и стопорного кольца и уплотнительного кольца в качестве второстепенных частей. Заглушка имеет место для закрытия манометра для определения давления внутри скважины. Внутри фланцевого корпуса имеется резьба API-UP TBG в качестве подъемной резьбы.
