Корпус катушек с трубными головками (PTHS) PARVEEN имеет прямое отверстие с посадочным буртиком под углом 45 градусов. В то время как верхний фланец имеет прижимные винты в стандартном оборудовании для крепления к трубе, нижний фланец предназначен для установки нижнего блока PARVEEN — от PBP-1 и PBP — 2 (уплотнение из полипропилена для высокого давления свыше 5000 фунтов на квадратный **дюйм** ) и направляющей насадки, портом для впрыска пластикового уплотнения с выпускным отверстием и контрольным портом. Это может принимать подвески для труб с оправкой типа «PTHSM», с обертыванием типа «PTHSW» и с подборкой горловины типа «PTHSE». Также могут быть предусмотрены даже 2-дюймовые трубопроводные и фланцевые выходы.Выходные отверстия с шипами имеют резьбу для заглушек для снятия клапана. Верхний фланец снабжен прижимными винтами для крепления трубодержателя.
