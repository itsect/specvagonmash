**А. Особенности конструкции**

1. Корпуса клапанов PARVEEN изготавливаются из поковок из легированной стали, которые различаются по различным торцевым соединениям, образуя их химические соединения.
2. Проба и вредность PARVEEN устойчива к истиранию и попаданию в организм.
3. Клапаны PARVEEN легко переносятся для гидравлического или пневматического привода. Эти оценки достигаются с полными коллекторными выбросами.
4. Вычисления в соответствии с API-6A и API-Q1.
5. Наш диапазон до 15000 PSI WP.

**Б. Принцип работы**

1. Пробка поворота на 90° (1/4 оборота) для быстрого полного открытия или закрытия. Это концентрация эрозии из-за дросселирующего действия.
2. Для раннего закрытия начального уплотнения корпуса клапана имеет конусообразную форму.
3. Для непрерывного использования становилось более эффективным по мере увеличения перепада давления, плунжера и вставки «плавают» вниз по потоку с перепадом давления, вызываемым первоначальным потреблением.
4. Соотношение между и зоной ведения таково, что крутящий момент, необходимый для работы клапана, минимален.
5. Предупреждение в упорном подшипнике для применения, плунжер уравновешенно подверженных поражению штока.
