Катушка подвески с прикрепленным коллектором рождественской елки затем опускается и герметизируется на втулке подвески. Теперь накидную гайку можно завинтить вручную с помощью гаечного ключа. Теперь дерево нельзя вращать. Поднимите сборку дерева достаточно, чтобы удалить паука и накладки. В этот момент скважина может быть связана с пакером, установленным под полным контролем давления. Затем завершаются операции по установке ниппеля, и обратный клапан типа PBPV снимается с втулкой и подвески с помощью лубрикатора.

**Фланцевая буровая катушка (PFDS)**

Они требуют срочной циркуляции бурового раствора в скважине и быстрой гибкости при размещении отверстий выкидных линий в соединениях противовыбросных превенторов. Могут быть предусмотрены шипованные или открытые концы и выходные соединения. Обычно ступица фланцевого патрубка приваривается. Высота шпули варьируется от 24″ до 36″. Однако также могут быть предоставлены катушки в соответствии с отклонениями от клиентов.

**Заглушки для снятия клапана (ПВРП)**

Все фланцевые или штифтовые выходы головок обсадных труб, катушек обсадных труб и головок насосно-компрессорных труб PARVEEN имеют резьбу для заглушек для снятия клапанов. Все заглушки имеют стандартную трубную резьбу API Line и достаточно малы, чтобы пройти через клапан, который необходимо удалить. Заглушка устанавливается, когда клапан должен быть снят, и снимается, когда клапан должен быть установлен.

**Инструмент для снятия клапана (ПВРТ)**

Как правило, это ручной гидравлический инструмент для сбалансированного давления, предназначенный для защиты и надежной работы по установке и снятию заглушек для снятия клапана, чтобы установить или заменить выпускной клапан устья скважины в условиях давления. Этот инструмент также предназначен для помещения 2 1 Клапан /16”, 2 9/16”, 3 1/8” до 10000 PSI WP. Адаптер может быть предоставлен для использования с фланцевыми клапанами 1 13/16”, 2 1/16”, 2 9/16”, 3 1/8” и клапанами с резьбовым соединением. Для подбора размеров заглушек также предусмотрены различные размеры гнезда для снятия клапана.
