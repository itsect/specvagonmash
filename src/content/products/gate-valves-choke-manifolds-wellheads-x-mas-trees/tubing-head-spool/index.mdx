Катушки НКТ PTHD для популярных подвесок с разъемными подвесками для двойного выпуска. Он имеет приваренные клинья для выравнивания подвески и открытый верхний фланец.

Трубная катушка типа 'PTHD' подходит для трубных подвесок 'PTHDE', 'PTHSCL', 'PTHDCL'. Эта катушка для снабжения труб клином для размещения подвески. Стандартные выходы 2 1/16 2000-10000 PSI WP с шипами. Выходные отверстия имеют резьбу для заглушки для снятия клапана, чтобы можно было снять клапан. Нижняя часть фланца связана с плотностью установки типа ПБП-1 (PP Seal для высокого давления более 5000 фунтов на квадратный дюйм WP) и снабжена нагнетательным портом пластиковой набивки и контрольным портом. верхняя часть катушки снабжена прижимными винтами для крепления трубки.
