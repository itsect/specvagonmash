Подвеска для трубы стержевого типа PARVEEN с уплотнениями металл-металл, называемая «PTHTSM», используется с трубной катушкой PARVEEN и расходом металла-металла для жидкостей, которые запрещают использование эластомерных уплотнений. Эти вешалки имеют металлические кольца вверху и внизу для герметизации. Диапазон условного рабочего давления составляет от 5000 фунтов на квадратный дюйм до 15000 фунтов на квадратный дюйм. Подвески PARVEEN PTHTSMM, оснащенные одинарными или задействованными гидравлическими линиями управления, оснащены PTHTSMM1 и PTHTSMM2. Они содержат металлические включения для жидкостей, запрещающие использование эластомерных уплотнений.На герметичность бросается в глаза металлическое кольцо, установленное на верхней части подвески, и нагрузка, приложенная к нижней части корпуса подвески.

**Адаптеры для трубных головок**

Адаптеры для трубных головок PARVEEN для подвески трубок и организации с подвеской с резьбой PTHSE с регулируемой горловиной для максимального рабочего давления 15000 фунтов на квадратный дюйм. Они могут быть снабжены шипованными соединениями и утопленным нижним фланцем с контрольными и выпускными отверстиями. Адаптеры для трубных головок также можно использовать с трубной подвеской (PTHSW) для макс. Рабочее давление 10000 PSI и внутренняя резьба для подвески НКТ. Он может иметь резьбу в соответствии с требованиями заказчика.
