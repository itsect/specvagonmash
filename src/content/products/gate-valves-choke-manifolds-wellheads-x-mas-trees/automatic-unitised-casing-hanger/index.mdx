**Автоматическая подвеска обсадной колонны** (PCHA) в основном связана с работой с нагрузкой и нагрузками и используется для корпуса обсадной головки PARVEEN «PCHH» и катушки обсадной головки «PCHS», описанных ранее. Когда нагрузка на обсадную трубу воздействует на подвеску, верхнее кольцо, подключаемое с помощью уплотнительного кольца и винта с буртиком, сжимает набивку, герметизируя кольцевое давление обсадной колонны. Эта подвеска обсадной трубы обычно используется для расчетного давления до 3000 фунтов на квадратный дюйм. Для прогнозируемого давления в 3000 фунтов стерлингов был использован тип «PSPO».

**Унифицированная подвеска для обсадной колонны** (PCHU) может использоваться как корпус головки обсадной колонны (PCHH), так и для катушки головки обсадной колонны (PCHS) PARVEEN, как обнаружен ранее. Эта вещалка встречается для среднего и незаметного обнаружения. Эта подвеска обычно защелкивается вокруг трубы и образуется в корпусе головки обсадной колонны. После посадки обсадной колонны винты с буртиками должны быть быстро затянуты, чтобы обеспечить надежное кольцевое использование. Подвеску можно использовать до 3000 PSI WP. Для прогнозируемого давления 3000 фунтов на квадратный дюйм используется PSPO, чтобы предотвратить разрушение трубы.
