title : "Кроссовер / разъемы"
---

Доступны в различных размерах, весах, встречаемости и классе с любой комбинацией соединений.

а. Диаметры — от 1,05 до 20 дюймов.

б. Вес – все учтены.

в. Длина – любая.

д. Марки – Н-40, Ж-55, К-55, Н-80, М-65, Л-80, С-90, С-95, Т-95, П-110, Q-125

е. Соединения — любая комбинация соединений Standard или Premium.

ПРИМЕЧАНИЕ. Соединения премиум-класса сдаются авторизованным подрядчикам по субподряду.

**НАРЕЗАНИЕ РЕЗЬБЫ:** PARVEEN является нарезчиком всех элементов группы 1, 2, 3 и 4 API 5CT.

**СОЕДИНЕНИЯ ДЛЯ ЛИНЕЙНЫХ ТРУБ**

а. Ном. Размер — от 1/8 до 12 дюймов.

б. Размеры — подтвердите размеры и допустите в соответствии с последней редакцией API 5L.

в. Резьба – Резьба муфт, методы измерения и проверки резьбы должны соответствовать API 5B.

д. Вес – муфты применимы либо к стандартному весовому, либо к сверхпрочному.

е. Маркировка - Муфты, изготовленные в соответствии с API-5L, будут маркированы в соответствии со спецификацией API-5. Индивидуальная маркировка также может быть предоставлена по запросу.

ф. Материал – материал API-5L.

**НАРЕЗАНИЕ РЕЗЬБЫ:** PARVEEN - это резьбонарезной станок для всех типов ЛИНЕЙНЫХ ТРУБ и аксессуаров, как определено в спецификациях API-5L. Подходит для труб от 1/8” до 12”

**ИНСТРУКЦИЯ ПО ЗАКАЗУ**

Для заказа кожухов, насосно-компрессорных труб, линейных труб, муфт, муфт и переходников используются:

Номинальный внутренний диаметр и внешний диаметр, толщина, толщина толщины, PP F, марка, типы соединений, особые требования и т.д. д.

ПРИМЕЧАНИЕ. Стандартная маркировка продукта соответствует спецификациям API 5CT/5L. Индивидуальная маркировка также может быть предоставлена по запросу.
