Данные сменные износостойкие вкладыши защищают внутренние полости обсадных колонных головок или катушек во время бурения.

Используются следующие износостойкие вкладыши:
- Для прижима с помощью винтов по верхнему фланцу корпуса колонной головки/головок. Имеет исполнение в виде прижимных винтов по верхнему фланцу колонной головки/головок.
- Для прижима с помощью винтов по отдельному фланцу, который будет установлен на корпус колонной головки/головок. Имеет исполнение в виде фиксирующего фланца.
- Для установки уплотнительного кольца на износостойком вкладыше. Имеет исполнение в виде уплотнительного кольца.
