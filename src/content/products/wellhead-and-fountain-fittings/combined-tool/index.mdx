---
title : "Комбинированный инструмент (Инструмент спуска/подъема износостойкого вкладыша и пробка для опрессовки ПВП)"
has_featured_image : true
has_pdf: false
---
Комбинированный инструмент спускается с ниппелем для установки или извлечения износостойкого вкладыша.

Инструмент спускается с ниппелем вниз для уплотнения по внутреннему диаметру гнезда для выполнения опрессовки блока ПВП или манифольда.

Бурильная труба может быть подвешена под опрессовочной пробкой, устраняя необходимость извлечения всей трубы из скважины.

Спуск выполняется на бурильной трубе.
