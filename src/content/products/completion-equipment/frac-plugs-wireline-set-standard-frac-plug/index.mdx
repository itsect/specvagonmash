Стандартная пробка для гидроразрыва обеспечивает экономичное средство временной изоляции зоны для гидроразрыва пласта или других видов обработки. Пробка может быть установлена на различных типах инструментов для установки давления на кабеле.
Стандартная пробка ГРП поставляется со срезным кольцом для обеспечения точной и надежной установки. В комплект поставки также входит шарик, который устанавливается на верхнюю часть пробки ГРП во время операции гидроразрыва. Эта пробка выдерживает умеренные давления и температуры.

**Характеристики:**

* Набор электрических тросов
* Разбуриваемый
* Чугунная конструкция
* Цельные клиновые плашки - закалены только на глубину плетения
* Комплектуется корпусом любой марки, в том числе П-110
* Нижний комплект со срезным кольцом
* Для временной или постоянной работы
* Стопорное кольцо с храповым механизмом удерживает установочное усилие
* Малый наружный диаметр для обеспечения скорости и безопасности при работе
