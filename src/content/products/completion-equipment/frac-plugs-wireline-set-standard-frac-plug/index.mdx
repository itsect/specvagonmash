Стандартная пробка для гидроразрыва используемое средство временной государственной зоны для гидроразрыва пласта или других видов обработки. Пробка может быть установлена на различных типах инструментов для установки давления на кабеле. Стандартная пробка ГРП подходит со срезным кольцом для обеспечения бесперебойной и надежной установки. Также в комплект поставки входит шар, который включает в себя часть пробки ГРП во время операции гидроразрыва. Эта пробка выдерживает умеренные давление и температуру.

**Функции:**

* Набор электрических тросов
* Разбуриваемый
* Чугунная конструкция
* Целевые плашки – закалены только на определенное плетение
* Комплектуется корпусом любой марки, в том числе П-110
* Нижний комплект со срезным кольцом
* Для временной или постоянной службы
* Стопорное кольцо с храповым механизмом удерживает установочное усилие
* Небольшие наружные диаметры для скорости и безопасности при беге
