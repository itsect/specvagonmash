Инструмент для извлечения пакера PARVEEN требуется для извлечения извлекаемого гидравлического пакера с отверстием под уплотнение. Каждый пакер имеет уникальный инструмент для извлечения, но все они имеют одинаковую базовую конструкцию. Фиксатор защелкивается на верхней резьбе пакера. Потяните за срезной винт цанговой опоры и освободите пакер. Если пакер не удается освободить с помощью правого вращения, цанга освобождается от предохранительной гайки, и защелка может быть отведена назад.Инструмент для извлечения пакера PARVEEN требуется для изъятия извлекаемого гидравлического пакера с отверстием под использование. Каждый пакер использует инструмент для изъятия, но все они имеют одинаковую базовую свечение. Защелка имеет защелку в верхней резьбе пакера. Потянув, срез срезной винт опоры цанги и восстановите пакер. Если пакер не может быть освобожден с помощью правого вращения, цанга будет свободна от контргайки, а защелка может быть выдвинута.

**Функции**

* Простое управление
* Безопасный вклад
* Прочная конструкция для многократного использования
