Мостовая гидравлическая пробка модели HM устанавливается с использованием электрической энергии для установки верхней планки, а затем механического натяжения для завершения установки. Премиум предназначен для высокого давления и температуры, в то время как Lite поддерживает умеренную температуру и давление. Необходимости в механическом инструменте для настройки не существует, потому что механизм настройки встроен. В колонну насосно-компрессорных труб вкладывается шар, перекрывающий встроенные уравнительные порты. Затем прилагается давление, чтобы зафиксировать верхнее скольжение, а затем приобретается механическое натяжение, чтобы закрыть закрепление.Освобождение колонны насосно-компрессорных труб от заглушек увеличенным поворотом вправо. Полный внутренний диаметр НКТ доступен после выхода из заглушки, что позволяет окружающему оборудованию проходить через конец колонны насосно-компрессорных труб.

**Функции:**

* Разбуриваемый
* Чугунная конструкция
* Цельный слип
* Комплектуется корпусом любой марки, в том числе П-110
* Металлическая опора для предотвращения экструзии резины
* Для временной или постоянной службы
* Стопорное кольцо с храповым механизмом удерживает установочное усилие
* Отлично подходит для наклонных скважин
