Цементная пробка модели HM устанавливается с использованием гидравлической энергии для установки верхних плашек, а затем механического натяжения для завершения установки. Класс Premium предназначен для высокого давления и температуры, а Lite поддерживает умеренную температуру и давление.
Необходимости в механическом установочном инструменте нет, поскольку установочный механизм встроен. В колонну насосно-компрессорных труб помещается шар, перекрывающий встроенные уравнительные порты. Затем прикладывается давление, чтобы зафиксировать верхнюю плашку, а затем применяется механическое натяжение, чтобы завершить закрепление. Освобождение колонны насосно-компрессорных труб от заглушки осуществляется простым поворотом вправо.
Полный внутренний диаметр НКТ доступен после освобождения от заглушки, что позволяет другому оборудованию проходить через конец колонны НКТ.

**Характеристики:**

* Характеристики:
* Разбуриваемый
* Чугунная конструкция
* Цельный клиновый захват
* Комплектуется корпусом любой марки, в том числе П-110
* Металлическая опора для предотвращения экструзии резины
* Для временной или постоянной работы
* Стопорное кольцо с храповым механизмом удерживает установочное усилие
* Отлично подходит для наклонных скважин
