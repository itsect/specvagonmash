Извлекаемый пакер PARVEEN TST-1 — это пакер для тяжелых условий эксплуатации, идеально подходящий для всех типов ремонтного цементирования, гидроразрыва пласта, кислотной обработки под высоким давлением и т. д. Это пакер, устанавливаемый давлением, с большим отверстием и гидравлической кнопочной фиксацией. Он выдерживает высокое давление сверху или снизу и использует трехэлементную систему уплотнения, L-образный паз и механизм тормозного блока для легкой настройки. Инструмент оснащен проверенной системой уплотнения из трех элементов и уравновешивающей втулкой с гидравлическим управлением, предназначенной для удержания перепускного клапана в закрытом состоянии при установке.

**ОСОБЕННОСТИ:**

* Надежно устанавливается в обсадной колонне любой твердости, в том числе премиум-класса
* Трехкомпонентные системы набивочных элементов с металлическими опорами подходят для применения при низком и высоком давлении для самого широкого спектра услуг.
* Гидравлический прижимной кнопочный блок с большим внутренним каналом потока, расположенный под перепускным клапаном, для уменьшения задирания элемента и затупления плетения кнопки. Эти прижимные клинья поршневого типа устанавливаются давлением снизу
* Комбинированный байпасный и уравнительный клапан с поверхностным управлением
* Автоматическая L-образная установка, установка пакера осуществляется спуском на глубину, вращением на 3/4 оборота вправо и установкой вниз
* Прямой захват НКТ сначала открывает перепускной клапан и выравнивает давление. Дальнейший захват отпустит пакер и автоматически переведет пакер в рабочее положение для дальнейших операций вверх или вниз по стволу скважины.
* Все резьбовые соединения входят в стандартную комплектацию резьбы API IF/EUE.
* Все уплотнительные кольца подшипников высокого давления для повышенной устойчивости к давлению
* Сопротивление нагрузке и давлению, разработанное для работы с высокими нагрузками
* Корпус, муфта и переводник P110 для тяжелых условий эксплуатации
