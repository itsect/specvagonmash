Пробка большого диаметра для гидроразрыва пласта предназначена для работ по гидроразрыву пласта, при которых требуется большое отверстие в пробке. Пробка поставляется со срезным кольцом для обеспечения точной и надежной установки. В комплект поставки также входит шарик, который устанавливается на верхнюю часть пробки ГРП во время операции гидроразрыва. Эта пробка выдерживает умеренные давления и температуры.

**Характеристики:**

* Набор электрических тросов
* Разбуриваемый
* Цельные клиновые плашки - закалены только на глубину плетения
* Комплектуется корпусом любой марки, в том числе П-110
* Верхний комплект со срезным кольцом
* Для временной или постоянной работы
* Стопорное кольцо с храповым механизмом удерживает установочное усилие
