Удлинитель уплотняющего отверстия PARVEEN используется в заканчиваниях, где пакер с уплотняющим отверстием используется в качестве эксплуатационного пакера и ожидается движение НКТ. Расширение отверстия уплотнения обеспечивает достаточную длину отверстия уплотнения, чтобы длинные узлы уплотнения могли компенсировать движение НКТ, всегда оставаясь в отверстии уплотнения. Несколько подпакерных седел уплотнителя можно соединить вместе с помощью концентрической муфты для увеличения длины отверстия уплотнения. Подпакерные седла уплотнителя доступны для всех размеров пакеров с уплотняющим отверстием.

Подпакерные седла уплотнителя изготавливаются из материалов, соответствующих стандарту NACE MR0175 STD. Для устойчивого к SSC металлического материала.

**Назначение**:
* Заканчивание эксплуатационными пакерами с герметичным отверстием там, где ожидается движение НКТ

**Преимущества**:
* Простота установки
* Недорогое решение
* Облегчает замену труб
