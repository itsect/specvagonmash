Роликовые штанги PARVEEN используются с бурильной колонной в наклонно-направленных скважинах. Они используются для минимизации трения, вызванного скольжением инструментальной колонны по стенке НКТ. Роликовые стержни представляют собой обычные тросовые стержни с прорезями, отфрезерованными для размещения простых роликовых колес.

* Другие размеры доступны по запросу.
* Доступная длина 2 фута, 3 фута, 5 футов
