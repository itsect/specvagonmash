Набор тросовых инструментов необходим для эффективного контроля над поверхностью во время спуска и протягивания каната / троса подповерхностных управляемых устройств.
Сборка тросовых инструментов, соединенных с тросом, используется для нанесения контролируемых ударов с поверхности (бурение) либо вверх, либо вниз, чтобы управлять устройствами в стволе скважины.

Стандартный набор инструментальной колонны обычно состоит из:
Веревочная муфта для крепления троса к инструментальной колонне.
Стержень штока / грузовой стержень для добавления веса для опускания инструмента в ствол скважины под действием давления скважины и встречающихся жидкостей различной тяжести.
Механический строп / ясс для фиксации забивного эффекта при движении вверх или вниз.
Дефлектор для обеспечения гибкости с помощью инструментальной колонны.
Инструмент для спуска/вытягивания для спуска и извлечения устройств из скважины.

Все инструменты доступны со следующим в соответствии с требованиями заказчика:
* Резьба насосной штанги
* Темы ООН
* Быстроразъемное соединение
Резьба насосной штанги нарезана на инструментах в соответствии с API 11B.
Резьба UN также обрабатывается на инструментах и взаимозаменяема с резьбой насосной штанги, но короче по длине.

Разъем с быстрым замком, как следует из названия, представляет собой быстроразъемное соединение, которое используется вместо резьбового соединения. Он имеет ряд преимуществ перед резьбовым соединением:
* Он включает соединение на четверть оборота.
* Это прочнее, чем резьбовое соединение. Он имеет три поверхности, несущие ударную нагрузку в каждом направлении, что делает его более безопасным для тяжелых и длительных операций.
* Конструкция более безопасна и проста и может быть освобождена с помощью отвертки.
