Набор тросовых инструментов необходим для контроля над поверхностью во время движения и протягивания каната / троса подповерхностных управляемых устройств.

Сборка инструментов на кабеле, подключаемых с кабелем, используется для применения кратковременных ударов с поверхности (сотрясение) либо вверх, либо вниз, чтобы манипулировать в стволе скважины.

Стандартный набор инструментальной колонны обычно состоит из:

**Веревочное гнездо** для крепления троса к инструментальной колонне.

**Стержень штока / грузовой стержень** для добавления вещества для погружения инструмента в стволы скважин против давления в скважинах и различных встречающихся газообразных жидкостей.

**Spang Link / Mechanical Jar** для быстрого ударного эффекта при движении вверх или вниз.

**Шарнирное соединение** для быстрой гибкости с помощью инструментальной колонны.

**Инструмент для взятия/вытягивания** для взятия и изъятия оборудования из скважины.

Все инструменты обратятся к согласованию с заказчиком

требования:

*  Резьба насосной штанги
*  Темы ООН
*  Соединение с быстрым замком

Резьба насосной штанги нарезана на инструменте в соответствии с API 11B.

Резьба ООН также сделала запрос на инструмент и взаимозаменяемость с резьбой насосной штанги, но короче по структуре.

Разъем с быстрым замком, как следует из названия, представляет собой быстроразъемное соединение и быстроразъемное соединение, которое используется вместо резьбового соединения. Он имеет ряд преимуществ перед резьбовым соединением:

*  Он включает соединение на четверть оборота.
*  Это прочнее, чем резьбовое соединение. Он имеет три поверхности, несущие ударную нагрузку, которую делает его вирус более опасным для обнаружения и длительных операций.
*  Это безопаснее и проще и может быть снято с помощью отвертки.
