Подъемный инструмент PARVEEN серии R используется для работ, при которых требуется сильный удар вниз. Инструмент предназначен для срезания и высвобождения путем удара вверх. Они доступны с четырьмя различными типами сердечников.

* RB: Длинный сердечник, короткий радиус действия.
* RS: Средний сердечник, средний радиус действия.
* RJ: Короткий сердечник, длинный радиус действия.
* RQ: Специальный сердечник, предназначенный для фиксации быстроразъемного соединения (QLS). Любой из этих инструментов можно изменить на другие типы, изменив сердечник. Все остальные детали остаются прежними.
