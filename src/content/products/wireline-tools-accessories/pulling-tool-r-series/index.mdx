Инструмент для вытягивания PARVEEN серии «R» используется для работ, при которых требуется сильное сотрясение вниз. Инструмент предназначен для разрезания и высвобождения путем сотрясения вверх. Они являются средними с распространенным типом сердечников.

1. РБ: Длинный сердечник, короткий радиус действия.
2. RS : Среднее получение, средний охват.
3. RJ: Короткий сердечник, большой радиус действия.
4. RQ: специальный сердечник, предназначенный для фиксации соединений Quick Lock Connection (QLS).

Любой из этих инструментов можно изменить на другие типы, изменив ядро. Все остальные деталишились прежними.
