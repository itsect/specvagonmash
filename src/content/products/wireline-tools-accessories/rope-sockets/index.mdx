Веревочные муфты PARVEEN используются для надежного крепления троса/каната к колонне инструмента. Они доступны в четырех типах.
* Тип клина: В клиновидном или грушевидном типе с втулкой используется латунный клин, чтобы удерживать трос, прикрепленный к веревочной муфте. Узел веревочной муфты состоит из корпуса с внешней ловильной горловиной, втулки и латунного клина.
* Тип катушки: Тип катушки или тип диска и пружины состоит из корпуса с внешней ловильной горловиной, пружины, опоры пружины и диска. Линия крепится к рифленому диску по его окружности.
* Тип клиновой плашки: Тип клиновой плашки предназначен для использования с небольшими плетеными струнами до 5/16". Устройство состоит из корпуса с наружной ловильной горловиной, каретки, плашек, установочного винта и донного переводника. Клиновые плашки предназначены для разрыва струны при определенном проценте прочности на растяжение струны.
* Тип зажима: Тип зажима состоит из корпуса с внешней ловильной горловиной, зажима, установочного винта и нижнего переводника. Струна помещается между зажимами и закрепляется установочными винтами.
* Другие размеры доступны по запросу
