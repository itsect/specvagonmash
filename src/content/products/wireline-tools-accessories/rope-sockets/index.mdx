Веревочные муфты PARVEEN используются для надежного крепления троса/троса к колонне. Они различаются в четырех типах.

1. Клиновидный тип: клиновидный или грушевидный тип с муфтой использует латунный клин, чтобы удерживать трос прикрепленным к тросовому гнезду. Узел Rope Socket состоит из корпуса с внешней ловильной горловиной, втулки и латунного клина.

2. Тип катушки: тип катушки или тип диска и пружины состоят из корпуса с внешней ловильной шейкой, пружины, опоры пружины и диска. Леска крепится к стрелковому диску по окружности.

3. Тип плашки: Тип плашки предназначен для использования с плетеными лесками до 5/16″. Он состоит из корпуса с наружной ловильной горловиной, каретки, плашек, установочного винта и донного переводника. Накладки процента для разрыва лески при высокой стойкости к заболеванию лески.

4. Тип замыкания: Тип замыкания состоит из корпуса с внешней шейкой, зажима, установочного винта и среднего перевода. Линия вставляется между зажимами и закрепляется установочными винтами.
