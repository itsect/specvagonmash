Инструмент переключения PARVEEN «B» используется для перемещения внутренней втулки в закрытое или открытое положение в скользящих втулках PARVEEN типа «PXD», «PXU», «PXA» и «CMD». Этот инструмент имеет два штифтовых соединения, позволяющих либо подключаться, либо закрывать втулку, переворачивая

**ИНСТРУМЕНТ ДЛЯ СЕЛЕКТИВНОГО ПЕРЕКЛЮЧЕНИЯ БО**

Инструмент селективного выбора PARVEEN 'BO' используется для выбора внутренней втулки скользящей втулки только в нижнем положении. Эти параметры определяют для насосно-компрессорных труб с многочисленными скользящими муфтами среднего размера, так что подходящую муфту можно сместить в нижнее положение без повторения других скользящих муфт.

**ИНСТРУМЕНТ ДЛЯ СЕЛЕКТИВНОГО ПЕРЕКЛЮЧЕНИЯ 'X'**

Инструмент выбора переключателя PARVEEN 'X' используется для выборочного перемещения внутренней втулки скользящей втулки в открытом или закрытом положении. Он отличается повышенной избирательностью в неизбирательном перемещении переключателя повышающе-вниз через втулку. Выбранную скользящую втулку можно сдвинуть в открытое или закрытое положение, не перемещая другие втулки. Собачки можно переместить в близкое положение к 90-градусному уступу ключа, подняв струну инструмента и снова опустив ее, чтобы циклически представить инструмент в положении. незафиксированное положение.
