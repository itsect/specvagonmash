Инструмент смещения PARVEEN B используется для перемещения внутренней муфты в закрытое или открытое положение в скользящих муфтах PARVEEN типа PXD, PXU, PXA и CMD. Этот инструмент имеет два штифтовых соединения, позволяющих открывать или закрывать втулку путем переворачивания инструмента.

**Инструмент селективного смещения**

Инструмент селективного смещения PARVEEN BO используется для выборочного смещения внутреннего рукава скользящей муфты только в нижнее положение. Эти инструменты предназначены для колонн НКТ с несколькими скользящими муфтами одинакового размера, так что нужную муфту можно сместить в нижнее положение без смещения других скользящих муфт.

**Инструмент смещения X**

Инструмент селективного смещения PARVEEN X используется для выборочного перемещения внутренней муфты скользящей муфты в открытое или закрытое положение. Он переключается из селективного в неизбирательное положение путем перемещения переключающего инструмента вверх-вниз через втулку. Выбранную скользящую муфту можно перевести в открытое или закрытое положение, не сдвигая другие муфты.
Собачки можно переместить в положение, близкое к 90-градусному уступу ключа, подняв струну инструмента и снова опустив ее, чтобы перевести инструмент в положение без фиксации.
