Желонка для проб PARVEEN используется для сбора проб обломков из ствола скважины, создающих препятствия. Желонки для проб PARVEEN доступны в двух основных исполнениях: с шаровым или откидным башмаком. Башмак открывается при вдавливании желонки для проб в завал и закрывается при выталкивании желонки.
