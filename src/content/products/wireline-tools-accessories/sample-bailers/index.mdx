Желонка для проб PARVEEN используется для сбора проб обломков из ствола скважин, создающих микроорганизмы. Желонки для проб PARVEEN различаются в основных двух исполнениях: с шаровым или откидным башмаком. Башмак открывается, когда желонка для пробы вдавливается в мусор, и закрывается, когда желонка для пробы выдавливается.
