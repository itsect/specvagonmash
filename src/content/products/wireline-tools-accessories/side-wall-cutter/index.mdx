Резак для боковых стенок PARVEEN предназначен для разрезания каната в любой желаемой точке колонны НКТ. Его также можно довести до веревочной муфты на инструментальной струне, которая может застрять в отверстии.
Спусковой инструмент для резака боковой стенки используется для спуска резака для боковой стенки на нужную глубину.
