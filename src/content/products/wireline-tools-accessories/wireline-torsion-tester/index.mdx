Тестер кручения троса PARVEEN представляет собой портативный настольный прибор для облегчения испытания и оценки качества троса из углеродистой стали в соответствии со спецификацией API 9A. Настоятельно рекомендуется проводить регулярные испытания троса в процессе эксплуатации, чтобы отслеживать неизбежное ухудшение свойств троса и предотвращать выход из строя троса в скважине. Тестер кручения троса предназначен для определения пластичности троса диаметром от 0,092 дюйма до 0,125 дюйма включительно. Испытание состоит из скручивания образца троса вдоль его собственной оси до отказа, при этом отмечая количество оборотов до отказа (сравните с API 9A для соответствия). Настоятельно рекомендуется проверить все канаты до начала скважинных работ.
