Спусковые инструменты моделей JK, RK-1 и JC-3 с соответствующими защелками используются для установки внутрискважинных устройств управления внутри выбранных оправок с боковыми карманами. Эти спусковые инструменты соединены с киковерами.

**Спусковой инструмент JK**: Спусковой инструмент JK используется с защелками BK и BK-2 для установки клапанов с наружным диаметром 1,00 дюйма и комплектующих в соответствующие оправки с боковыми карманами.

**Спусковой инструмент RK-1**: Спусковой инструмент RK-1 используется с защелками RK, RK-1, RKP для установки клапанов с наружным диаметром 1,50 дюйма и комплектующих в соответствующие оправки боковых карманов.

**Спусковой инструмент JC-3**: Спусковой инструмент JC-3 используется с защелками R и RA для установки клапанов с наружным диаметром 1,50 дюйма и комплектующих в соответствующие оправки боковых карманов.
