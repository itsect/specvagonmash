Спусковые инструменты моделей «JK», «RK-1» и «JC-3» с одинаковым защелками используются для установки внутрискважинных устройств управления внутри выбранных оправок боковых карманов. Эти спусковые инструменты сочетаются с инструментами для отбрасывания.

**Спусковой инструмент «JK»:** Спусковой инструмент «JK» используется с защелками BK и BK-2 для установки клапанов с наружным диаметром 1,00 дюйма и аксессуарами во время оправки боковых карманов.

**Спусковой инструмент «РК-1»:** Спусковой инструмент «РК-1»используется с защелками РК, РК-1, РКП для установки клапанов с наружным диаметром 1,50 мм и аксессуарами с наружной стороны о боковые карманы.

**Инструмент для применения «JC-3»:** Инструмент для применения «JC-3» используется с защелками R и RA для установки клапанов с наружным диаметром 1,50 дюйма и приспособлениями во время оправки боковых карманов.
