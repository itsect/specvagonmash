Подъемный инструмент PARVEEN серии S используется для зацепления и извлечения оборудования с внешней ловильной шейкой путем удара вверх. Инструмент предназначен для срезания и освобождения путем удара вниз. Благодаря этой функции инструмент также можно использовать в качестве спускового инструмента. Они доступны с тремя различными типами сердечников.

* SB:	Длинный сердечник, короткий радиус действия.
* Нержавеющая сталь:	Средний сердечник, средний радиус действия.
* SM:	Предназначен для использования в газлифтных операциях.
