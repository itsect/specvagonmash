Перфоратор для труб PARVEEN представляет собой механическое устройство, предназначенное для перфорации стандартных и толстостенных труб под давлением. При развертывании в скважине традиционными способами перфоратор не требует взрывчатых веществ, поэтому исключается возможность перфорации обсадной колонны.

**К преимуществам конструкции относятся**:
* Увеличенное проникновение НКТ
* Положительный признак перфорации при удалении инструмента
* Механизм безопасного выпуска, позволяющий извлекать инструмент из скважины без перфорации.
* Перфоратор предназначен для извлечения перфоратора и автоматического выпуска после перфорации.
* Использует направленные вверх удары для перфорации

Пробойник перфоратора используется в сочетании с трубной муфтой или стопором клинового типа, который обеспечивает точку крепления для активации инструмента.
