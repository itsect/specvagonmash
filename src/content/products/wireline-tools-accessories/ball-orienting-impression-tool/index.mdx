Шариковая печать PARVEEN — это вспомогательный инструмент, который используется вместе со свинцовой печатью для обследования скважины Parveen (LIB) во время ловильных работ. LIB используется для снятия слепков от посторонних предметов в колонне НКТ. Сделанный слепок определяет форму и ориентир для определения положения обструкции. Шариковая печать для обследования скважины при использовании с LIB более точным образом определяет упавший предмет. Вторичный свинцовый отпечаток (свободно движущегося) шарика внутри инструмента используется по отношению к основному отпечатку на LIB. Отпечаток шара обеспечивает точку отсчета относительно упавшего предмета. Рассчитав угол между двумя отпечатками, можно получить более точную картину положения упавшего предмета.
