Копья PARVEEN Bull Dog Spears — это рыболовный инструмент для обнаружения условий эксплуатации, предназначенный для внутреннего зацепления трубчатого предмета внутри колодца. Копье Bull Dog состоит из верхней вспомогательной части с рыболовной шейкой, штифтовым соединением и конической шпонкой, оправки, скользящей шпонки и обычного перевода. Закаленные шлицевые ключи имеют место быть типа «ласточкин хвост» для работы в условиях обнаружения. Дополнительные обжимные стопорные кольца создаются по запросу.
