Стальные тросовые штанги PARVEEN — это рыболовный инструмент для тяжелых условий эксплуатации, предназначенный для внутреннего зацепления трубчатого предмета внутри скважины.
Стальная тросовая штанга состоит из верхней вспомогательной части с ловильной шейкой, штифтовым соединением и конической шпонкой, оправки, скользящей шпонки и нижнего переводника. Закаленные шлицевые ключи имеют конструкцию типа «ласточкин хвост» для работы в тяжелых условиях. Дополнительные обжимные стопорные кольца могут поставляться по запросу.
