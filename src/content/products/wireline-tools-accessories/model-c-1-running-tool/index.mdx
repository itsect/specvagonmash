Спусковой инструмент PARVEEN C-1 используется для запуска устройств контроля потока PARVEEN, которые имеют наружную ловильную шейку. Он имеет нижнее болтовое соединение для крепления хвостовика модели A или N-1.
Хвостовик модели А используется для удержания защелок замков моделей S, W и Z в задвинутом положении во время спуска. Стойка модели N-1 используется для удержания защелок во втянутом положении в замках модели G и R во время спуска.
Спусковой инструмент C-1 оснащен защитой резьбы для применений, не требующих блокировки на самом инструменте. Чтобы преобразовать спусковой инструмент C-1 в спусковой инструмент непроходного типа, вместо протектора резьбы устанавливается стопорное кольцо.
