Резчик провода PARVEEN используется для разрезания каната рядом с гнездом, когда скважинные инструменты застряли и не могут быть извлечены из скважины. Резчик провода опускается вниз под собственным весом, направляясь по струне. Резчик перерезает проволоку, когда она бьет по веревочной муфте. Затем резчик провода вылавливается из скважины стандартным ловильным инструментом.
