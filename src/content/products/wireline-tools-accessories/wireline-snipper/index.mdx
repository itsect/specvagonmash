Ножницы для троса PARVEEN используют для разрезания троса рядом с гнездом троса, когда скважинные инструменты застряли и не собрали извлеченные из скважины. Ножницы для троса опускаются вниз по своему усмотрению, направляясь по леске. Ножницы перережут проволоку, когда она ударит по гнезду веревки. Затем щипцы для троса вылавливаются из скважин стандартными рыболовными инструментами.
