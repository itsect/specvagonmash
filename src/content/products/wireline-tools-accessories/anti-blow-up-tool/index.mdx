Инструмент PARVEEN Anti Blow Up Tool представляет собой инструмент для обслуживания на кабеле, предназначенный для использования в качестве части набора инструментов, когда скважинные инструменты должны быть изучены в скважине с многочисленными зонами.

Устройство Anti Blow Up Tool поможет предотвратить взрыв бурильной колонны в эксплуатационной колонне, когда скорость между вихрями имеет достаточную силу, чтобы вытолкнуть бурильную колонну вверх.

Когда нижняя часть струны инструменты начинают подниматься, два рычага инструмента выбрасываются наружу, чтобы зафиксироваться в стенке трубы, останавливая любое дальнейшее движение вверх. Чтобы разблокировать руки, вы просто берете трос, который поднимает инвестиционный инструмент и закрывает руки, освобождая инструмент.

Инструмент Anti Blow Up Tool подходит для труб размером от 2-3/8” до 5-1/2”.
