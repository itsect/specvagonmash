Шарнирное соединение для датчиков памяти PARVEEN используется для облегчения безопасного и эффективного крепления датчиков памяти к колонне инструментов. Оно изготовлено из нержавеющей стали для минимизации коррозии. Не рекомендуется, когда необходимо сильное ударное действие.
