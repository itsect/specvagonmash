Подъемные инструменты PARVEEN серий JD и JU используются для захвата и извлечения оборудования с внешними ловильными шейками. Подъемный инструмент серии JD предназначен для срезания и высвобождения путем удара вниз. Эта функция позволяет использовать подъемный инструмент серии JD в качестве спускового инструмента. Подъемный инструмент серии JU предназначен для срезания и высвобождения путем удара вверх.

Эти инструменты доступны с тремя различными типами сердечников:
* JDC/JUC: Длинный сердечник, короткий радиус действия.
* JDS/JUS: Средний сердечник, средний радиус действия.
* JDL/JUL: Короткий сердечник, длинный радиус действия.
Любой из вышеперечисленных типов можно преобразовать в другой тип, изменив сердечник. Все остальные детали остаются прежними.

* Радиус действия — это расстояние от корпуса до зацепляющего плеча собаки.
* Другая длина сердечника / внешний диаметр доступны по запросу.
