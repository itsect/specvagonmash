Подъемный инструмент PARVEEN для тяжелых условий GS представляет собой подъемный инструмент с полным радиальным контактом для сильного удара и для захвата плохо очищенных ловильных шеек. Он также может извлекать изношенную или поврежденную ловильную шейку благодаря площади контакта с собачками на 35 % больше, чем у стандартного ловильного инструмента GS.
