Спусковой инструмент PARVEEN W устанавливает в НКТ шлицевые шпиндели типа B, C и W. Спусковой инструмент W состоит из верхнего переводника в комплекте с ловильной шейкой и соединением штифта насосной штанги, среднего переводника, нижнего переводника, набора цанг и внутреннего сердечника.
При достижении требуемой глубины установки инструменты поднимаются для установки шлицев и расширения резинового элемента. После установки штифт в W-образном подъемном инструменте может быть срезан, а зубец выведен из цанги, что позволяет освободить спусковой инструмент из шпинделя.
