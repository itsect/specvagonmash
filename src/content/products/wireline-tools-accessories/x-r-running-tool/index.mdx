Спусковые инструменты PARVEEN X и R используются для установки стопорных шпинделей моделей X, XN, R, RN и RQ в соответствующие посадочные ниппели. Спусковой инструмент X и R можно настроить для работы в выборочном или неизбирательном режиме. В выборочном режиме стопорную оправку можно пропустить через несколько посадочных ниппелей одного размера и типа, прежде чем выбрать нужный ниппель для установки стопорной оправки. В неизбирательном режиме стопорная оправка найдет первый соответствующий ниппель.
