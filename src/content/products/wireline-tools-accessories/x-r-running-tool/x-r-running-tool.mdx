---
title : "Инструмент для запуска X & R"
---

Спусковые инструменты PARVEEN X и R используются для установки стопорных шпинделей моделей «X», «XN», «R», «RN» и «RQ» в посадочных ниппели. Инструмент запуска X и R можно настроить для работы в выборочном или неизбирательном режиме. В выборочном алгоритме стопорный стержень можно пропустить через несколько посадочных ниппелей одного размера и типа, чем прежде выбрать нужный ниппель для установки стопорного стержня. В неизбирательном режиме стопорная оправка найдет первый подходящий ниппель.
