Подъемный инструмент PARVEEN GS предназначен для разблокировки и извлечения различного скважинного оборудования с внутренними ловильными горловинами. Эти инструменты можно высвободить из устройства путем удара вниз. Установив переходник на сдвиг GU, подъемный инструмент GS можно преобразовать в инструмент для извлечения ясса (подъемный инструмент GR).
