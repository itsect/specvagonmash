Желонка для песка PARVEEN используется для удаления песка и другого мусора, осевшего на поверхности подземного оборудования, препятствующего подъему оборудования с помощью регулярных операций на тросе. Это инструмент насосного типа, который работает по принципу обычного подъемного насоса и поршня. Желонки с насосом для песка доступны со следующими типами нижних башмаков:

* С плоским дном	: Для легкого вычерпывания песка.
* С угловым дном	: Для выкапывания твердо утрамбованного песка
* С нижней частью заслонки	: Для сбора металлических частиц, которые не могут пройти через шар и седло.
