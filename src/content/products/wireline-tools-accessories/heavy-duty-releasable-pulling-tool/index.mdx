Съемный подъемный инструмент для тяжелых условий эксплуатации PARVEEN представляет собой подъемный инструмент цангового типа, используемый для фиксации и извлечения канатных инструментов с сильно поврежденными внешними ловильными шейками.
Сверхмощный съемный подъемный инструмент имеет чрезвычайно прочную конструкцию и позволяет защелкнуть ловильную шейку на все 360 градусов.
Ловильный инструмент можно высвободить, потянув вниз, при условии, что имеется прочная опора для сердечника, по которому можно ударить.
