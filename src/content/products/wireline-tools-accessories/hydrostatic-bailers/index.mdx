Гидростатическая желонка PARVEEN используется для удаления мусора, осевшего на поверхности подземного оборудования, которое извлекает извлекаемое оборудование с помощью регулярных операций на тросе. Гидростатическая желонка используется, когда мусор не может быть удален из скважины с помощью желонки с насосом для песка.
