Инструмент для проверки установки PARVEEN X используется для проверки правильности установки стопорных оправок X и R в соответствующий посадочный ниппель. Срезанный штифт в инструменте после его запуска указывает на то, что замок установлен правильно. Он опускается в скважину на стопорную оправку с помощью стандартных канатных инструментов.
