Инструмент для проверки PARVEEN X используется для проверки правильности установки стопорных оправок «X» и «R» в подходящий посадочный ниппель. Вырезанный штифт в инструменте после его запуска на то, что замок установлен правильно. Он встречается в скважине на стопорную опору с использованием стандартных канатных инструментов.
