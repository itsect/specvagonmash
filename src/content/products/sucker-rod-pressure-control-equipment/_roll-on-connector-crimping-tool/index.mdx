title : "Инструмент для обжима роликовых коннекторов"
has_featured_image : true
has_pdf: true
---

Инструмент для обжима роликовых коннекторов использует простую установку роликового коннектора PCE на гибкую трубу в полевых условиях. Обжимной инструмент имеет два взаимозаменяемых колеса, одно из используемых для обжатия гибкой трубы в накатной коннектор. Другой - отрезной круг, который можно использовать для резки гибкой трубы.
