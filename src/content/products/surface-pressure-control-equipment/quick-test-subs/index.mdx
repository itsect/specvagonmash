Переводник для экспресс-теста PARVEEN предназначен для экономии времени буровой установки при испытании лубрикатора под давлением, прежде чем подвергать его воздействию давления в скважине. При многократном спуске троса лубрикатор приходится проверять каждый раз, когда нарушается нижнее соединение, что приводит к потере значительного времени. После проведения первой опрессовки всей колонны последующие опрессовки можно проводить с помощью переводника для экспресс-теста, чтобы проверить целостность отсоединенного соединения, а не испытывать весь лубрикатор в сборе. Это делается путем подключения небольшого ручного насоса к переводнику для экспресс-теста для быстрой проверки двух уплотнительных колец снаружи.
Переводник для экспресс-теста устанавливается либо выше, либо ниже противовыбросового устройства и становится соединением, которое отсоединяется каждый раз для ввода и извлечения инструментов из скважины.
