Заглушки нижнего байпаса PARVEEN модели G доступны в следующих размерах:

**FSG** — Работа со всеми ниппелями модели F
**FWG** — Работа с верхними непроходными ниппелями модели F
**RZG** — Работа с нижними непроходными ниппелями модели R

Заглушки этого типа устанавливаются в перепускном положении, чтобы обеспечить прохождение скважинного флюида через компоновку при посадке оборудования в ниппельный профиль. Для спуска заглушки используется спусковой инструмент C-1.
После установки в профиль ниппеля эта группа заглушек может удерживать давление сверху и снизу.
Давление выравнивается перед извлечением путем вытягивания выравнивающей оправки.
Для вытягивания узла заглушки используется стандартный подъемный инструмент и соответствующий щуп для типа замка.
Заглушки нижнего байпаса PARVEEN G изготавливаются для стандартных условий работы с H2S-CO2.

**Область применения:**

* Выбранные зоны могут быть обработаны или закрыты
* Для подачи испытательного давления на трубопровод
* Для изоляции НКТ при ремонте или демонтаже устья скважины
* Для установки гидравлических пакеров
* Для ввода или вывода насосно-компрессорных труб из скважины

**Информация для заказа:**

Пожалуйста, укажите модель ниппеля, размер отверстия уплотнения, модель плунжера, рабочее давление и температуру, процентное содержание H2S и CO2.
