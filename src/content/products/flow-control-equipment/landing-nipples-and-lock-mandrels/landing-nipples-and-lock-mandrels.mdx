**Посадочные ниппели и стопорные оправки X и R от PARVEEN**

Посадочные ниппели X и R PARVEEN спускаются в скважину на насосно-компрессорных трубах, чтобы обеспечить специальное место для посадки оборудования для контроля притока. Общие внутренние профили этих посадочных ниппелей делают их универсальными. Посадочный ниппель X PARVEEN используется в трубках стандартного веса. Посадочный ниппель R PARVEEN обычно используется с тяжелыми трубами.
На колонне НКТ может быть установлено столько селективных ниппелей с одинаковым идентификатором, сколько необходимо. Такая универсальность обеспечивает неограниченное количество позиций для установки и фиксации органов управления подповерхностным потоком. Регулятор потока, прикрепленный к требуемому стопорному шпинделю X или R PARVEEN, спускается в скважине с помощью селективного спускового инструмента на тросе.
Оператор каната с помощью селективного спускового инструмента может установить регулятор потока в любом из посадочных ниппелей на желаемой глубине. При неудовлетворительном расположении или при изменении условий эксплуатации скважины регулятор потока может быть перемещен вверх или вниз по колонне НКТ в другое место расположения ниппеля. Эти операции могут быть выполнены с помощью канатно-проволочных работ под давлением без глушения скважины.

**Посадочные ниппели и стопорные оправки XN и RN PARVEEN - непроходные**

Это оборудование предназначено для использования в установках с одним ниппелем или в качестве нижнего ниппеля в серии посадочных ниппелей PARVEEN X или R. Эти посадочные ниппели имеют одинаковый внутренний диаметр сальникового отверстия для трубок определенного размера и веса. Посадочные ниппели PARVEEN X и XN предназначены для использования с трубами стандартного веса. Посадочные ниппели PARVEEN R и RN предназначены для использования с тяжелыми трубами.
(буква N обозначает непроходные ниппели)

**Характеристики**

* Посадочные ниппели
* * Большое отверстие для минимального ограничения
* * Универсальный ниппель с одним внутренним профилем
* Замковые оправки
* * Выдвижные запирающие ключи
* * Замки, предназначенные для блокировки давления сверху или снизу от резкого разворота
* Дополнительное удержание
* * Удержание помех для небольших замков
* * Удерживайте срезной штифт для больших замков
