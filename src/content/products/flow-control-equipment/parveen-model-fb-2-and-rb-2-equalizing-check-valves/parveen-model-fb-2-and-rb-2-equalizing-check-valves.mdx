Уравнительные обратные клапаны PARVEEN моделей FB-2 и RB-2 представляют собой оборудование для заканчивания без каких-либо запирающих устройств. Они используются в следующем оборудовании, смонтированном на НКТ:

**«FB-2»** – Работайте со всеми ниппелями модели F и со скользящими муфтами всех моделей L
**«RB-2»** – Работа с нижними непроходными ниппелями модели R

Обе модели входят в профиль ниппеля и держат давление только сверху. Модель FB-2 приземляется на верхнюю часть отверстия уплотнения ниппельного профиля F. Модель RB-2 устанавливается на нижний непроходной буртик ниппеля R.
Спусковой инструмент C-1 используется для спуска обоих клапанов в сборе.
Обе модели можно выровнять перед извлечением, открыв порты выравнивающей оправки. Для извлечения этих клапанов используется стандартный подъемный инструмент.
Уравнительные обратные клапаны PARVEEN FB-2 и RB-2 изготавливаются для стандартных условий эксплуатации, а также для работы с H2S и H2S-CO2 .

**Область применения:**

* Может использоваться в качестве заглушки для испытания трубок под давлением
* Для установки гидравлического пакера с обратным клапаном, расположенным под пакером
* Для газлифтных работ
* Для использования в качестве запорной арматуры в скважинах со скважинными электронасосами.

**Информация для заказа:**

Пожалуйста, укажите модель ниппеля, размер отверстия уплотнения, модель обратного клапана, рабочее давление и температуру, процентное содержание H2S и CO2.
