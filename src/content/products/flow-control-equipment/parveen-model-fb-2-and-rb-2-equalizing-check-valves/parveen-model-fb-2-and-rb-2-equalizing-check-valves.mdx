Уравнительные обратные клапаны PARVEEN моделей «FB-2» и «RB-2» используют собственное оборудование для выпуска без каких-либо запирающих устройств. Они могут пользоваться в следующем оборудовании, установленном на НКТ:

**«FB-2»** **используется во всех ниппелях модели «F» и во всех** **скользящих муфтах модели «L».**

**«РБ-2»** **бежит по нижним непроходимым** **соскам «Р»**

Обе модели имеют профиль ниппеля и держатель давления только сверху. Модель «FB-2» приземляется на остановке Ниппельного профиля «F». Модель «RB2» установлена на нижнее непроходное плечо ниппеля «R».

Спусковой инструмент «C-1» используется для отдельных частей клапанов в сборе.

Обе модели можно выровнять перед изъятием, открыв порты выравнивающей оправки. Стандартный съемник используется для изъятия клапанов.

Уравнительные обратные клапаны PARVEEN 'FB-2' и 'RB-2' изготавливаются для стандартных условий, HS и HS-CO 2 2 2.

**Приложения:**

1. Может использоваться в качестве заглушки для трубок для испытаний под давлением.

2. Установите гидравлический пакер с обратным клапаном, расположенным под пакером.

3. Для газлифтных работ.

4. Для использования в качестве запорной арматуры в скважинах, оборудованных глубинными электронасосами.

**Информация для заказа:**

Пожалуйста, выберите модель ниппеля, размер концентрации, модель обратного клапана, напряженность давления и температуру, процентное содержание HS и Co. 2 2
