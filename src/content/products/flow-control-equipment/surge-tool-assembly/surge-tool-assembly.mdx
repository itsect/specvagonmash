**Применение**

* Инструмент PARVEEN для пластовых перенапряжений должен быть укомплектован соответствующим замком/уравнительным узлом. Сборка должна быть спущена в скважину обычным способом, а устройство должно быть размещено в соответствующем ниппеле. Спусковой инструмент должен быть извлечен до использования инструмента для повышения давления в пласте.

**Преимущества**

* Спроектирован таким образом, что через перфорацию создается тяга для удаления мусора
* Может быть извлечен с помощью обычных операций на кабеле после стабилизации давления в скважине.
