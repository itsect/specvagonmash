**Приложение**

* Формовочный инструмент PARVEEN Surge Tool должен быть собран с соответствующим замковым/выравнивающим узлом. Сборка должна быть спущена в скважину обычным способом, устройство должно быть размещено в исследовании ниппеле. Инструмент для рассмотрения должен быть использован до использования инструмента для повышения давления в пласте.

**Преимущества**

* Вызванные для возникновения возможности создания просадки через перфорацию для удаления мусора.
* Может быть извлечен с помощью обычных операций на кабеле после измерения давления в скважине.
