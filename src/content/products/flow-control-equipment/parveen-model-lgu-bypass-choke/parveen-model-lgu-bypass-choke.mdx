БАЙПАСНЫЙ ДРОССЕЛЬ ТИПА LGU представляет собой верхнее непроходное устройство, которое работает на кабеле и предназначено для спуска и установки в скользящую муфту типа T, идеально подходящую для смешанной добычи. БАЙПАСНЫЙ ДРОССЕЛЬ оснащен керамическим дросселем, который регулирует поток в зоне производства через скользящую муфту. Продукция из другой зоны поступает через байпас, а обратный поток через дроссель и скользящую муфту предотвращается обратным клапаном с шаром и седлом по стандарту API.
