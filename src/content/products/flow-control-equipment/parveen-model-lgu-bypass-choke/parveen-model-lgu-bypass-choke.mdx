БАЙПАСНЫЙ ДРОССЕЛЬ ТИПА «LGU» представляет собой собственное устройство Top No-Go, которое работает на кабеле и предназначено для конкретной установки в скользящую муфту типа «L», идеально подходящую для смешанной добычи. БАЙПАСНЫЙ ДРОССЕЛЬ оснащен керамическим дросселем, который регулирует поток в зоне наблюдения через скользящую муфту. Продукция из внешней зоны проходит через байпас, ному обратному потоку через штуцер и скользящую втулку через шар API и обратный клапан со спинкой седла.
